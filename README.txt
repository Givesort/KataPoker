Running:
	Open the project in eclipse or your java IDE
	Run KataPokerMain

Using:
	Enter player one's hand (looks like "ah kh qh jh th") without duplicating cards
	Press enter
	Enter player two's hand without duplicating cards or re entering cards in player one's hand
	Press enter
	See the output
	Repeat

Stopping:
	Type "exit" (without quotes)
	Press enter
